This project template is the basis of my ARM projects.

The base project schematic includes:

  • The "BrechtVE_Aesthetics", "BrechtVE_DebugHeader" and "Sparkfun-PowerSymbols" symbol libraries
  • The CERN OHL v. 1.2. page layout
  • The OSH logo
  • Four M3 holes
  • A modified 0.05" 2x05 ARM Cortex-M SWD header with added UART functionality
  • A TagConnect 2050 programming header for the same SWD/UART functionality
  • A 100 kΩ pullup resistor on SWDIO
  • A 4-pin 0.1" UART header with 100 Ω inline current limiting resistors and a 10 kΩ pullup resistor on RXD
  • A reset button with a 100 Ω current limiting resistor to GND, 1 kΩ pullup resistor and 100 nF debounce capacitor
  • PWR flags


The base project PCB layout includes:
  • The "BrechtVE_Aesthetics" and "SparkfunConnectors" footprint libraries
  • The default Text & Graphics settings
  • Modified design rules for the Net Classes, Tracks & Vias and Solder Mask/Paste
